# speckle-app-test

# PoC - SF Speckle Param Updater 


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```

## credentials

In order to run the project, create a .env file with the following env variables

VITE_APP_SPECKLE_ID=
VITE_APP_SPECKLE_SECRET=
VITE_APP_SERVER_URL=
VITE_APP_SPECKLE_NAME=
